Gets the Warframe market information for one item. Items will always return an entire set for any item.

Note: Warfrmae market has a certian layout for item urls, it's recommneded to get the url from MarketClient.getMarketItems() in order to get that url.

Future<ItemSet> getMarketItem(String item) async {
  final payload = await _client.get('/items/$item');

  return ItemSet.fromJson(payload['item'] as Map<String, dynamic>);
}
